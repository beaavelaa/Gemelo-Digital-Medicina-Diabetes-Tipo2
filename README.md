# Gemelo Digital en Medicina

## Simulación de Pacientes Virtuales mediante Machine Learning para la Predicción y Prevención de la Diabetes Tipo 2

### Descripción

Proyecto desarrollado en el ámbito de la salud digital y la medicina personalizada mediante la creación de un gemelo digital capaz de simular pacientes virtuales y estimar el riesgo de desarrollar Diabetes Mellitus Tipo 2.

### Dataset utilizado

Pima Indians Diabetes Database

Fuente:
- National Institute of Diabetes and Digestive and Kidney Diseases (NIDDK)
- UCI Machine Learning Repository

### Objetivos

- Analizar factores de riesgo asociados a la diabetes.
- Desarrollar modelos predictivos mediante Machine Learning.
- Simular pacientes virtuales.
- Apoyar la toma de decisiones clínicas.

### Tecnologías utilizadas

- C#
- Visual Studio
- Machine Learning
- Análisis de Datos
- Inteligencia Artificial aplicada a la salud
